삼각함수
cosA = a/c
sinA = b/c

a = cosA * c
b = sinA * c

임의의 점 (x,y) 가 있을때
30도 에 100만큼 떨어져있는 좌표를 구할때

(x + a , y + b)
(x + cosA * c,y+sinA * c)
(x + cos30 * 100,y + sin30 * 100)

cos0 = 1
sin0 = 0

오른쪽 방향

sin90 = 1
서있는 선

라디안 호도법
컴퓨터 프로그래밍에서 전부 라디안으로 표기
호의길와 현의길이가 같을때가 1라디안
호가 원이되면 2ㅠ 라디안

2ㅠ = 360
ㅠ = 180

변환법
180 -> *ㅠ/180 -> ㅠ
ㅠ -> *180/ㅠ -> 180
